# EZselect
### It's as easy as
1. ```npm i ezselect```
2. ```const ez = require('ezselect')```
3. ```const ezTitle = ez('title')```

### Functions

| Function                  | Usage                                          |
|---------------------------|------------------------------------------------|
| .css("Property", "Value") | Set the CSS of an element.                     |
| .on("Event", Function)    | Add an event listener to an element.           |
| .el                       | Returns a reference to the element in the DOM. |
